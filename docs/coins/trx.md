# Tron

This is a Tron Driver for the Platform SDK. The implementation makes use of [Tron](https://tron.network/) and adheres to the contracts laid out in the [specification](/docs/specification.md). **This package should not be used directly. It is designed to be consumed through the [SDK](/docs/sdk.md) package.**

## Installation

```bash
pnpm install @ardenthq/sdk-tron
```

## Security

If you discover a security vulnerability within this package, please send an e-mail to [security@ardenthq.com](mailto:security@ardenthq.com). All security vulnerabilities will be promptly addressed.
